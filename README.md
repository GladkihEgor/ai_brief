# ai_brief
Базовый курс по машинному обучению, дающий представление об основных методах и алгоритмах ML

## Материалы использовались для курсов:
1. Теория машинного обучения и искусственного интеллекта
2. Искусственный интеллект для физиков

## Структура курса

7 лекций
14 практик


## Темы практических работ:
1. разведочный анализ
2. классификация ч.1.
3. классификация ч.2.
4. регрессия
5. кластеризация
6. понижение размерности
7. неслабансированный набор (фрод) ч.1.
8. неслабансированный набор (фрод) ч.2.
9. нейронные сети, енкодер-декодер архитектура
10. NLP
11. подкрепление
12. Финальный проект и его защита

## Финальный проект
Проектная группа - 2-3 человека (НЕ БОЛЬШЕ)		
Название проекта (либо придумываете, либо придумываем вместе)		
Ссылка на датасет (если датасет закрытый - обсуждаем отдельно)		
Репозиторий - на GitHub		
		
Требования к проекту		
1. Самостоятельное решение		
2. Достаточно сложная задача		
3. Презентация по результату проекта		
	Название	
	Постановка задачи	
	Предобработка данных	
	Выбор моделей для обучения	
	Получение метрик	
	Выводы	
	.= доступный код, который выполняется.	
		
4. Доклад - 10-15 минут + вопросы
