## Финальный проект
Проектная группа - 2-3 человека (НЕ БОЛЬШЕ)
Название проекта (либо придумываете, либо придумываем вместе)
Ссылка на датасет (если датасет закрытый - обсуждаем отдельно)
Репозиторий - на GitHub

Требования к проекту
1. Самостоятельное решение
2. Достаточно сложная задача
3. Презентация по результату проекта:
	- Название
	- Постановка задачи
	- Предобработка данных
	- Выбор моделей для обучения
	- Получение метрик
	- Выводы
	- доступный код, который выполняется.

4. Доклад - 10-15 минут + вопросы
